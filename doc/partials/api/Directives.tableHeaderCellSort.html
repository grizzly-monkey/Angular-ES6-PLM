tableHeaderCellSort
service in module Directives

Description

To set the sort order on the sortable table headers.
This should be used with Directives.tabularDataWidget and Directives.tableColumnSort.

  • data

    The column data.

  • sortOrder

    The sort order - can either be 'asc' or 'desc'.

  • doSort

    The sort function from the respective controllers.

Dependencies

  • Requires Controllers.TabularDataWidgetController
