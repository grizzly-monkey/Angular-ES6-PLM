tableColumnResize
service in module Directives

Description

A feature of tabular data widget to support column resizing. This should be used with TabularDataWidgetDirective. The concept is based on ng-grid.

Dependencies

  • Requires Controllers/Controller.TabularDataWidgetController

Usage

as attribute
<ANY table-column-resize>
   ...
</ANY>

Methods

  • getAdjacentColumnIndex(columnList, startIndex)

    Find an index of the adjacent 'non-fixed' column based on the given start index.

    Parameters
    ParamTypeDetails
    columnListObject[]

    : List of columns

    startIndexNumber

    : index from where the search will begin.

    Returns
    Number/null

    index of the adjacent column. Null in case no column found.

  • mouseMove()

    mouse move event handler. It will update the position of the resize overlay.

  • mouseUp()

    mouse up event handler. It will update the width of the columns.

  • triggerInit()

    Initialization function

Properties

  • columnHeaderElementList

    list of column header elements.

  • currentColumnIndex

    keep the current column index during resizing process.

  • resizeOverlay

    a resizeOverlay element that will be added to the table component when resize is in-process.

  • resizerCurrentX

    keep the current position of the resizer during resizing process.

  • resizerList

    list of resizers associated with the table header columns.

  • resizerStartX

    keep the starting position of the resizer when resizing process started.

  • tableLeft

    reference of the left position of the table component.
