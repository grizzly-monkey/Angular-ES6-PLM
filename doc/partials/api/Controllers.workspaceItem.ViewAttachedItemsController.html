ViewAttachedItemsController
service in module Controllers.workspaceItem

Description

Display list of attached items for a single item.

Methods

  • cancelSelection()

    Close the flyout.

  • goToAttachment()

    go to attachment tab of the item

Properties

  • attachmentsCount

    The total count value of attachments.

  • flyoutState

    This property is used with Directives/AdjustMaxHeightDirective to allow on-demand max height calculation.

  • goAttachmentsLink

    build the link to attachment tab of the item, require for opening in new tab

  • itemId

    The item id of the current selected item.

  • workspaceId

    The workspace id of the current selected item.
