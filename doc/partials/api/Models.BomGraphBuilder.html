BomGraphBuilder
service in module Models

Description

A service that handles the building of a BomGraph.

Methods

    Build an edge and node given the collection of properties

    | | <- Edge (Edge Created, All properties populated) | V (Child Node) <- Node (Node Created, All properties except OutEdges populated)

    We parse the BomNestedItem edgeNodeProperties back onto our edges/nodes and add them to the graph

    ParamTypeDetails
    `edgeNodeProperties`BomNestedItem

    needing to be parsed to created an edge node

    `viewDef`Object

    the current view definition that is to be used in building the properties of the edge and the node fields.
