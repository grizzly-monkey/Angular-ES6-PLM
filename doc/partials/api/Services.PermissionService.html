PermissionService
service in module Services

Description

A utility service that contains permission related methods.

Dependencies

Methods

  • checkPermissionByItem(itemObj, permission, applyLock:, applyRevisionOverrideLock)

    Evaluates whether user has permission on the workspace or not on the basis of given item object.

    Parameters
    ParamTypeDetails
    itemObjObject

    : workspace item object.

    permissionNumber

    : the permission to check

    applyLock:Boolean

    an optional parameter that will be used only when getting workspace id using $stateParams#itemId. This parameter (if provided) will control whether we need to apply extra workspace locking check or not.

    applyRevisionOverrideLockBoolean

    : an optional parameter that will be used only when getting workspace id using $stateParams#itemId. This parameter (if provided) will control whether we need to enforce extra permissions or not.

    Returns
    Promise

    a promise that resolves with true if user has permissions.
