BulkBomLoader
service in module Models

Description

A service that is used to load the BulkBom.

Methods

  • constructor()

    Intializes the service object.

  • loadBulkBom(`eventId`, `bulkBomEndPointLink`, `queryParameters`)

    makes a request for a Bulkbom using the bulkBomEndPointLink and queryParameters.

    Parameters
    ParamTypeDetails
    `eventId`String

    The unique id that will be used to broadcast the request for loading the BulkBom. (We are using the urn of item).

    `bulkBomEndPointLink`String

    the endpoint link for the bulk bom.

    `queryParameters`Object

    the parameters that needs to be passed to the endpoint in order to make the request properly.

    Returns
    Promise

    A promise that eventually resolves to the fetched BulkBom.

  • requestForBulkBom(`eventId`, `itemId`, `queryParameters`)

    Parameters
    ParamTypeDetails
    `eventId`String

    The unique id that will be used to broadcast the request for loading the BulkBom. (We are using the urn of item).

    `itemId`String

    the bulk bom endpoint link.

    `queryParameters`Object

    the query parameters to be sent to the end point.
