WorkflowTransitionController
service in module Controllers

Description

This controller is responsible for managing the workflow transition flyout. It is used with Controllers.ItemHeaderController.

Methods

  • cancelTransition()

    Closes the workflow transition flyout.

  • isCommentsRequiredAndInvalid()

    True, if required comments have been filled in.

  • itemTransitionsListenerId()

    private Helper method to retrieve a list of transitions (DRY)

  • proceedTransition()

    Check the validity of the form and closes the flyout if valid.

  • setChosenActAsUser(Will)

    It sets the values of the chosen act as user

    Parameters
    ParamTypeDetails
    WillString

    be the string passed while selecting the user

  • userListenerId()

    private Helper method to retrieve user object

Properties

  • UserObj

    The user object

  • hasDelegators

    True, if there are available delegators

  • maxCommentsLength

    The maximum allowed length for the comments.

  • minWidth

    A min width of the form. The minimum width will be based on the comments type and anchor width.

  • transition

    The workflow transition associated with this controller.

  • workflowTransitions

    The possible transitions.
