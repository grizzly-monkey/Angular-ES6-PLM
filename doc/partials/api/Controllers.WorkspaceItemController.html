WorkspaceItemController
service in module Controllers

Description

This controller corresponds to /workspaceItems/{workspaceId}. It represents an abstract state and contains nested views.

Dependencies

Methods

  • getStateName()

    Gets the state name for the tab, with its params

  • isStateActive(activeStateId, stateViewId, stateEditId)

    Checks which is the current active state

    Parameters
    ParamTypeDetails
    activeStateIdString

    The name of the active state passed by the view

    stateViewIdString

    The name of the view state passed by the view

    stateEditIdString

    The name of the edit state passed by the view

    Returns
    Boolean

    true/false If it's the current state

  • viewLoad()

    Called whenever a new view has been loaded (an item's "tab"), and broadcasts an event

Properties

  • itemId

    The item id of the current selected item.

  • tabList

    Stores parsed list of tabs for the current item

  • tabMapping

    private The mapping of icons and ids for the tabs The 'icon' is not being used in Galileo, but left here for legacy purposes in case we use them later

  • tabsListenerId

    private The listener for the loading of the tabs

  • workspaceId

    The workspace id of the current selected item.
