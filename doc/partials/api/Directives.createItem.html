createItem
service in module Directives

Description

This directive is used to create a workspace item. - formFields Object The fields displayed on the dialog - filter Object Function to filter the type of fields - unsupportedFields Object TO BE REMOVED - selectedWorkspace Object The current workspace context - workspacesList Array List of workspaces for which the create item will be allowed - createType String The type of create - waiting Boolean If the data is in the process of being POSTed to the backend

Dependencies

Usage

as attribute
<ANY create-item>
   ...
</ANY>
