EditAdditionalOwnersController
service in module Controllers.workspaceItem

Description

This controller is used for controlling the behavior of editAdditionalOwners.html

Dependencies

Methods

  • closeDialog()

    closes the change owner dialog

  • triggerSave()

    triggers a save to update values of current owners.

Properties

  • groups

    available groups that the user can add as additional owners

  • modified

    true if there is a difference between the orginal and selected users and groups

  • originalGroups

    groups part of the original additional groups

  • originalUsers

    users part of the original additional users

  • selectedGroups

    groups that are currently selected

  • selectedUsers

    users that are currently selected

  • users

    available users that the user can add as additional owners
