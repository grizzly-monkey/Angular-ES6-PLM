BomChangePinningEdit
service in module Models

Description

A change that edit the pinning of an item Either this.revisionField and this.currentRevision or this.targetItem must be defined for the change to be used

Methods

  • constructor()

    Initializes the edit

  • isRevertingChange()

    Determines if the change is a reverting change If the pinning has changed, it is not reverting if the revision field is provided, and the current revision has changed from the original revision it is not reverting otherwise, it is reverting

Properties

  • currentRevision

    The currently selected revision Has the property 'title', which uniquely identifies the revision and 'version', which contains all the revision's information

  • revisionField

    The field corresponding to the revision

  • targetItem

    The item which should be pinned This is ignored if revisionField and currentRevision have been provided since those allow us to determine the isRevertingChange state of the selected revision, instead of just relying on checking if the pinning changed or not
