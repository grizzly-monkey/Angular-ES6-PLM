fieldViewTruncation
service in module Directives

Description

This directive is to be used inside fieldView, just to display a truncated value

- shouldTruncate {Boolean} If the truncation should happen
- value: {String} the plain string
- length: {Integer} the length
- preformatted: {Boolean} If should wrap the string using 
 instead of , to respect "\n"
- html: {Boolean} If should display the passed string as HTML

Dependencies

Usage

as attribute
<ANY field-view-truncation>
   ...
</ANY>
