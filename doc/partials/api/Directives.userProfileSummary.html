userProfileSummary
service in module Directives

Description

This directive displays a specific user's profile

Dependencies

    • Requires Services.UserProfileService

Usage

as element:
<user-profile-summary>
</user-profile-summary>

Properties

  • avatarSrc

    The URL of the user's avatar image

  • data

    User profile data

  • profileItems

    Specifies the items to be displayed in the user profile and their respective icons

  • userObj

    The Promise resolving to user object providing information about the user
