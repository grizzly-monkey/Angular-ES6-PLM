ReportChartController
service in module Controllers

Description

This controller is responsible for logics of report chart directive

Dependencies

  • Requires Services.LocalizationService

Methods

  • closeChartModal()

    Closes the modal window when clicking the close button

  • createCategoryList(axisId, chartData)

    create list of categories from the given chart data.

    Parameters
    ParamTypeDetails
    axisIdString

    : the id of the axis on which the category will be created. Note: Generally for reports it will be x-axis.

    chartDataObject

    : the chart data for the report.

    Returns
    Array

    list of categories for the chart.

  • createFieldDescriptor(field)

    create field descriptor on field object.

    Parameters
    ParamTypeDetails
    fieldObject

    : field associated with the report.

    Returns
    String

    the descriptor of the field based on fieldID and optionally aggregate properties. Note: we need to incorporate aggregate as reportResult column id will have the aggregate property as part of field ID.

  • createId(field)

    create field id based on field object.

    Parameters
    ParamTypeDetails
    fieldObject

    : field associated with the report.

    Returns
    String

    the id of the field based on fieldID and optionally aggregate properties. Note: we need to incorporate aggregate as reportResult column id will have the aggregate property as part of field ID.

  • createInitialDataSetForSeries(initialValue, length)

    create default data list for a series. This method will be used with createSeries.

    Parameters
    ParamTypeDetails
    initialValueNumber

    : initial value to be set to the list.

    lengthNumber

    : the length of the list. Note: In the context of series, this should be the length of categories.

    Returns
    Array

    list with default values.

  • createSeries(seriesMap, categoryList, dependentAxisId, independentAxisId, defaultValue)

    create list of series of the report.

    Parameters
    ParamTypeDetails
    seriesMapObject

    : a map keyed by series name and value is the list of values for categories. Note: list does not necessary contain value for each category, so this method will fill the missing values with default values.

    categoryListArray

    : list of strings representing the category for the report chart.

    dependentAxisIdString

    : the dependent axis id (it can be x-axis or y-axis depending on the chart data)

    independentAxisIdString

    : the independent axis id (it can be a-axis or y-axis depennding on the chart data).

    defaultValueNumber/String

    : the default value to set to the series data list.

    Returns
    Array

    the list of series for the report chart.

  • createSeriesMap(seriesId, chartData)

    create a map of series from the chart data.

    Parameters
    ParamTypeDetails
    seriesIdString

    : the id of the series (this will be the key of the resultant map).

    chartDataObject

    : the chart data for the report.

    Returns
    Object

    map of series keyyed by series id NOTE: its valid to have an empty string as series id. What should we do if we have multiple series with empty string id??

  • init()

    Initialization function

  • init()

    Initialization function

  • isMultiSeriesOrStacked()

    true, if chart is either multiseries or stacked

  • useTicks()

    true, if ticks will be used on x-Axis (independent axis) instead of categories. For line and area charts, ticks should be used.

Properties

  • bundle

    localization bundle

  • chartConfig

    The chart configuration option to be used with high charts directive. Note: Need to set some default values to avoid showing default title and other options when chart config is being loaded.

  • isRendered

    true, when report chart directive is rendered.
