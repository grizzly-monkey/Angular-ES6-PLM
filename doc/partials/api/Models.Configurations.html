Configurations
service in module Models

Description

A Model for the configurations stored at api/v3/configurations

Methods

  • constructor()

    Parses the json and stores it in a map

  • fetch()

    Fetches the configurations

  • getConfig(key)

    Returns a config objected associated with the key, or undefined if there is no associated config

    Parameters
    ParamTypeDetails
    keyString

    The key to lookup

    Returns
    Object

    the config object

  • getConfigKey()

    Returns a key by which the config object can be accessed Specifically, '/api/v3/configurations/helpLocation' will return 'helpLocation'

Properties

  • constructor

    A map of configuration keys to configuration objects

  • json

    The json used to build this object
