infiniteScroll
service in module Directives

Description

This directive handles infinite scrolling It expects the following attributes/parameters:

- update: a function to be called on scroll to the end of current list
- context: an optional parameter bounding the scroll event. if this is not set, the scroll even will be binded to the element, if this is set, assume the context is window
- infiniteScrollDisabled: an optional parameter that can be used to conditionally disable infinite scroll.

Dependencies

Usage

as attribute
<ANY infinite-scroll>
   ...
</ANY>

Methods

  • conditionMet()

    A private helper method that tests for scroll

  • conditionMet()

    A private helper method that tests for scroll with context

  • scrollHandler()

    A private helper method to check for infinite scroll condition.
