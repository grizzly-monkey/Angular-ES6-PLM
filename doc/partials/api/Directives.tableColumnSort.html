tableColumnSort
service in module Directives

Description

To get the default sort column of tabular data widget.
This should be used with Directives.tabularDataWidget.

Dependencies

  • Requires Controllers.TabularDataWidgetController

Usage

as attribute
<ANY table-column-sort>
   ...
</ANY>

Methods

  • doSort()

    Determines the current column to be sorted and calls the sort function in the respective controllers.

  • triggerInit()

    Initialization function

Properties

  • columnHeaders

    private The list of table header, corresponds to tableColumns in TDWCtrl.

  • columnHeadersElements

    private The list of actual table header elements.

  • columnScopeMap

    The map to trace each column to its respective column scope.

  • currentSortColId

    The current column that is being sorted.

  • sortConfig

    The sort configuration from respective controllers.
    disableInitialSort: True, to disable the initial sorting and remove the sort carets.
    applySort: The actual sort function from the respective controllers.
