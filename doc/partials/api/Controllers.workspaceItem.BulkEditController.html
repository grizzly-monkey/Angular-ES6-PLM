BulkEditController
service in module Controllers.workspaceItem

Description

This controller responsible for handling the modal window for bulk edit This controller takes the selected items from the affected items view and fetched the state id and thereafter the corresponding transitions stateLifecycle is an array in the controller that contains the stateObj which inturns has the object contains the state name and the corresponding avilable transitions

Methods

  • closeBulkTransition()

    Responsible for closing the bulk edit modal overlay

  • effectivityChanged()

    Used for disabling and enabling the date field

  • getAffectedItemsByState(stateURN)

    Get affected items by state urn. This method uses itemMap to map state urn to affected item

    Parameters
    ParamTypeDetails
    stateURNString

    the urn of the state.

    Returns
    Array

    the list of affected items based on the given state urn.

  • updateBulkTransition()

    update selected affected items based on the selected lifecycle and effectivity options. It also closes the dialog and passes the updated affected items to the parent component.

Properties

  • InvalidDateValidation

    It is responsible for showing the invalid date validation

  • distinctStateList

    It holds the distinct state list

  • effectivity

    an object to hold the data related to effectivity date.

  • itemMap

    A map of workspace item keyed by item urn.

  • itemPromiseList

    It holds the list of promises of itemObj

  • meta

    an object to hold metaData the field selector

  • stateLifecycle

    It holds the complete lifecycle stateinformation including the state title and transitions

  • stateList

    It holds the state list
