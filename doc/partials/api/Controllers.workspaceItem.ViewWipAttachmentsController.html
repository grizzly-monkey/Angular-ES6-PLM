ViewWipAttachmentsController
service in module Controllers.workspaceItem

Description

View Controller for WIP enabled attachments view

Methods

  • isViewState()

    Determine whether we're in view mode

    Returns
    Boolean

    true if we are in view mode

  • setSelectedFiles()

    Sets the selected files

  • triggerCancel()

    State change triggered by clicking cancel button.

    NOTE: name has to be triggerCancel.

  • triggerDownload()

    Send an event to CPDM to trigger file download.

  • triggerEdit()

    State change triggered by clicking edit button.

    NOTE: name has to be triggerEdit.

  • triggerSave()

    State change triggered by clicking save button.

    NOTE: name has to be triggerSave.

Properties

  • addPermission

    The add attachments permission.

  • attachmentsData

    The reference of attachment grid's data. This property is binded with grid directive.

  • dateFormat

    Stores the user-defined date format.

    date: Date format, defaults to 'yyyy'.
    dateAndHour: Date-plus-hour format, defaults to 'yyyy hh:mm a'.

  • deletePermission

    The delete attachments permission.

  • editPermission

    The edit attachments permission.

  • isItemLoaded

    True, if the attached items are loaded.

  • isItemLocked

    True, if the current item is locked.

  • itemId

    The item id of the current item.

  • itemLink

    The resource id of the current item.

  • itemListenerId

    Listener for item object. Determines the workspace id. Also updates the user permissions.

  • itemUrn

    The item urn (not encoded).

  • selectedFiles

    The selected WIP attachments.

  • triggerFileBrowserDialog

    Triggers a dialog for WIP file browser.

  • triggerRemove

    Trigger a broadcast remove event

  • viewPermission

    The view attachments permission.

  • workspaceId

    The workspace id of the current item.
