ViewDefinitionFieldListeners
service in module Models

Description

A class that registers the listeners for interacting with the Bom View Definition field endpoint

Methods

  • constructor()

    Registers the listeners

  • registerGetListener()

    Registers the get listener recieves 'viewDefinitionField:(urn):get', with a link to the view definition field

    Returns
    String

    the unique id of the listener

  • respondToGetListener(event, viewDefFieldLink, Other)

    responds to the event by calling the fetch method of the viewDefinitionField endpoint viewDefFieldLink with the parameters args. Then sends'viewDefinitionField:(fieldId):done' message with an instance of ViewDefinitionField

    Parameters
    ParamTypeDetails
    eventObject

    the event that was sent to request the view definition field

    viewDefFieldLinkString

    end point link for fetching the field

    OtherArray

    query parameters needed in order to make the call.
