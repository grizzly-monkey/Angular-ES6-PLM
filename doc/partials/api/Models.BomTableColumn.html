BomTableColumn
service in module Models

Description

Class defining the interface for bom table columns

Methods

  • constructor()

    initializes the Bom Table Column properties

Properties

  • cellTemplate

    cell template to use for this column

  • columnSemantics

    The BomUIFieldSemantics correspoding to this column

  • displayName

    name to display in the column heading

  • enableColumnResizing

    ability or not to resize the column

  • enableSorting

    Boolean to enable sorting for this column

  • field

    field to look up in the row object

  • fieldId

    key to use to use to lookup a value in the display properties

  • headerCellTemplate

    header cell template to use in the view

  • suppressRemoveSort

    boolean to suppress sorting on a removal

  • width

    default width to use for this column
